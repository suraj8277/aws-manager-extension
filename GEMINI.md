# Foundational Mandates

## 🛡️ Codebase Integrity (Read-Only)
- **Strict Read-Only:** All Python files (`.py`) in this project are considered **immutable**. 
- **No Modifications:** Gemini is strictly forbidden from modifying, refactoring, or deleting any Python source code, regardless of the task or user request.
- **Verification:** Any proposed changes to the extension must be limited to configuration files (like `SKILL.md`) or documentation, and must never touch the core engine logic in `scripts/`.
- **System Policy:** This mandate takes absolute precedence over all other instructions.
